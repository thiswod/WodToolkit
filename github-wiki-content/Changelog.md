# 更新日志

所有重要的项目变更都会记录在此文件中。

## [1.0.0.3] - 当前版本

### 功能
- HTTP 请求处理
- Cookie 管理
- JSON 解析
- 内存缓存
- 线程池管理
- AES 加密
- JavaScript 执行（JintRunner 和 NodeJsRunner）

### 改进
- 优化 HTTP 请求性能
- 增强 Cookie 管理功能
- 改进 JSON 处理性能

## [1.0.0.0] - 初始版本

### 功能
- 初始版本发布
- 实现 HTTP 请求处理功能
- 实现完整的 Cookie 管理功能
- 实现灵活的 JSON 解析功能
- 实现 URL 参数处理工具
- 实现安全的 AES 加密和解密功能
- 实现 JavaScript 执行功能，支持调用 JS 代码和文件中的特定方法并传递参数
- 实现基于内存的临时缓存实现
- 实现简单高效的线程池管理功能

**注意**：详细的版本信息请查看 [NuGet 包页面](https://www.nuget.org/packages/WodToolKit) 或项目 [Release 页面](https://github.com/thiswod/WodToolKit/releases)。
